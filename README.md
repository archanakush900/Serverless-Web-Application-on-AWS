# Serverless-Web-Application-on-AWS
Developed a serverless web application using AWS Lambda, Amazon DynamoDB, and Amazon S3. The application enables users to perform Create, Read, Update, and Delete (CRUD) operations on data stored in DynamoDB, providing a scalable and cost-effective cloud-based solution.

****Project Implementation****
Create a DynamoDB table to store application data.
Develop an AWS Lambda function to perform Create, Read, Update, and Delete (CRUD) operations.
Host the frontend files (HTML, CSS, and JavaScript) in an Amazon S3 bucket.
Configure Amazon CloudFront to securely deliver the web application with improved performance and reduced latency.


****Project Outcome****
A fully functional serverless web application deployed on AWS.
Practical experience with AWS Lambda, DynamoDB, S3, and CloudFront.
Understanding of how different AWS services work together to build a scalable and cost-effective solution.
Improved knowledge of cloud computing concepts, serverless architecture, and AWS service integration.
